1. Description

Creates a scrolling text from a givenn text.

2. Inputs

No. Name Initialisation Description
1 E1 [Text] ""
The complete text to be shown as scrolling text.
2 E2 [Updated (seconds)] 1
Interval for updating the scrolling text
3 E3 [Steps (Characters)] 1
Steps used every interval
4 E4 [Scrolltext length] 14
The length of the output of the scrolltext
5 E5 [Start Position] 0
Starting position on which the output should start
6 E6 [On/Off] 0
On/Off of the scrolltext

3. Outputs

No. Name Initialisation SBC Description
1 A1 [Text] "" s
The created scroll text
s = send, sbc = send by change
